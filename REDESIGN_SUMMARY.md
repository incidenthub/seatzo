# SEATZO Redesign Walkthrough

The project has been transformed into a **premium minimalist** platform. The design is inspired by high-end Awwwards-winning websites, focusing on bold typography, ample negative space, and smooth interactions.

## 🎨 Visual Identity & Design System

- **Typography**: 
  - **Syne**: Used for a bold, cinematic heading experience.
  - **Inter**: Used for clean, legible body text and interface elements.
- **Color Palette**: 
  - **Primary**: Pure Black (`#000000`) and Pure White (`#ffffff`).
  - **Secondary**: Stone Grays (e.g., `#f5f5f7`) for subtle depth.
  - **Accent**: Solid Black for CTAs and status indicators.
- **Components**: Glassmorphism, ultra-slim borders, and large rounded corners (`3rem`).

## 🚀 Core Redesign Highlights

### 1. The Entry (Home & Navigation)
- **Cinema Hero**: High-impact staggered text reveal and decorative floating elements.
- **Marquee**: A scrolling text section for dynamic movement.
- **Slim Navbar**: Transparent-to-glass transition on scroll with a minimalist "SZ" mark.

### 2. High-End Flow (Public Pages)
- **Curation Gallery (Listings)**: A premium grid layout with staggered entries and category-based filtering.
- **Cinematic Detail**: A split-screen layout featuring a sticky cinematic poster and a sophisticated progress-tracked availability system.

### 3. Secure Transactions
- **Auth Reimagined**: Split-screen Login and Register pages with high-contrast forms and clean typography.
- **Checkout Command**: A refined, security-focused checkout experience with real-time fee breakdowns.
- **Digital Voucher**: A ticket-style confirmation card with minimal QR placeholders and action shortcuts.

### 4. User Command Center
- **The Dashboard**: A "Pro" command center feel with a stats grid and clean, row-based booking history.

## 🛠 Tech Implementation
- **Tailwind V4 CSS Variables**: Centralized styling via `index.css`.
- **Framer Motion**: Staggered reveals, smooth transitions, and magnetic-hover effects.
- **Lucide Icons**: Unified minimalist icon set used consistently across all modules.
